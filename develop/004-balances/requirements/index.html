    Balances¶

    Situation¶

    There are many situations where account balances for a given account and date are required, and this is not a feature in the version 2.0.6 release.

    Proposal¶

    Add a new feature to get balance for a given account and date.

    Design¶

    The AccountTrans table is a ledger which contains all transactions for all accounts. To get the balance for a given account and date, sum the amounts of all transactions for that account from the balance date (transType=0), to the query date.

    AccountTrans

    • key: unique key of the record
    • accountKey: key of the account
    • timeStamp: time that the record was last updated
    • transType: coded type of transfer 0 (balance), 1 (expense), 2 (income), 3 (transfer out), 4 (transfer in)
    • transKey: key of the transaction, which can be used to get the transaction details from either Expense,Income, Account or Transfer tables.
    • transDate: transaction date
    • transAmount: amount in currency of the account
    • checked (not used)

    transType

    code sign type
    0 + balance
    1 - expense
    2 + income
    3 - transfer out
    4 + transfer in

    To get the balance for a given account and date, sum the amounts of all transactions for that account from the balance date (transType=0), to the query date.

    reconcile balance

    The reconcile balance is a reconciliation set by the user that can occur at any time from the beginning of the account to the present. So it is possible to have transactions both before and after the reconcile balance. Queries for balances prior to the reconcile balance will be based on subtracting the sum of transactions from the reconcile balance amount, while queries for balances after the reconcile balance will be based on adding the sum of transactions to the reconcile balance amount.

    There is only one reconcile balance record for each account.

    get reconcile balance sql query example

    get the latest reconcile balance and date for account 114

    select
    transDate
    ,transAmount
from AccountTrans
where accountKey = 114 AND
transType = 0
;
